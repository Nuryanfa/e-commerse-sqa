# 🥬 SayurSehat — E-Commerce Sayur Segar

SayurSehat adalah platform web inovatif yang menyediakan layanan *marketplace* khusus untuk jual-beli sayur mayur dan bahan pangan segar. Proyek ini menjembatani berbagai pihak mulai dari pengguna umun, pengepul, hingga pengantar untuk memudahkan jalur distribusi komoditas secara efisien.

---

## 🏗️ Tech Stack

Secara umum, platform ini dibangun menggunakan perpaduan teknologi terkini yang memberikan performa dan kecepatan respon terbaik:
- **Backend:** Go (Golang)
- **Frontend:** React (Vite) & Tailwind CSS
- **Database:** PostgreSQL

---

## 🚀 Petunjuk Instalasi

Aplikasi ini ditujukan untuk eksekusi server lokal (Localhost). Berikut langkah konfigurasinya:

### 1. Persiapan Repositori
```bash
git clone https://github.com/Nuryanfa/e-commerse-sqa.git
cd e-commerse-sqa
```

### 2. Konfigurasi Backend (Golang)
Pastikan Anda memiliki koneksi Database PostgreSQL yang telah dikonfigurasi.
```bash
cd backend-go
# Salin dan sesuaikan .env jika dibutuhkan
go run cmd/api/main.go
```

### 3. Konfigurasi Frontend (React)
Pastikan `Node.js` dan paket `npm` telah terpasang di sistem operasi Anda.
```bash
cd frontend-react
npm install
npm run dev
```
Aplikasi antarmuka akan berjalan dan dapat diakses melalui antarmuka web melalui tautan yang dimunculkan `http://localhost:5173`.
